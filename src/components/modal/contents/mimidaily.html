개발목표
  • - 팀 구성 및 역할 분담을 통한 웹 애플리케이션을 개발

담당역할
  • - PM, 서기
  • - 회원가입/정보수정 파트 담당
  • - TOP10 페이지 담당

구현기능
  • - MVC 패턴 기반의 구조 설계 (Servlet + JSP + Java)
  • - JDBC를 통한 Oracle DB 연동 및 DB 구성
  • - 기본적인 CRUD를 통한 회원/게시판 관리 기능 구현
  • - JavaScript를 이용한 프론트 유효성 검사 및 동적 처리
  • - 사용자 편의를 위한 비동기 처리
  • - 반응형 기반 UI 구성(가급적 순수 CSS 사용)
  • - 관리자 계정 포함한 사용자 유형 구성(비회원 / 일반 / 기자 / 관리자)
  • - 최신기사는 날짜순으로 정렬
  • - 카테고리(여행지 / 맛집)별 기사는 조회수 높은순으로 정렬
  • - 좋아요 수를 기준으로 인기 기사 Top10을 실시간 정렬하여 별도 페이지에 출력
  • - 기사 검색 기능 (제목 / 내용 기준)을 최신기사 / 여행지 / 맛집 페이지별로 각각 구현
